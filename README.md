# AI Retail Billing Web App – Brunda Traders

A web-based retail POS (Point of Sale) system developed for **Brunda Traders**, a grocery retail business that sells rice, dal, and cooking oil products.
This application helps manage **product inventory, billing, sales tracking, and reports** in a simple and user-friendly interface.

## Key Features

### Secure Login System
The application provides a login page for authorized users to securely access the POS dashboard.

### Dashboard Analytics
The dashboard displays important business insights including:
- Today's Sales
- Bills Generated
- Top Selling Product
- Low Stock Alerts
- Visual sales charts
  
<img width="1865" height="910" alt="Image" src="https://github.com/user-attachments/assets/90bc99e8-4ecc-4ffd-ad41-8dc78478c854" />

### Billing System
The billing interface allows the shop owner to:
- Search products
- Add products to cart
- Adjust quantity
- Automatically calculate total price
- Complete customer sales quickly
  
<img width="1907" height="783" alt="Image" src="https://github.com/user-attachments/assets/16ada118-a0a6-4588-bcc3-d32a487b9d89" />
### Product Management
Users can manage store products easily:
- Add new products
- Edit existing products
- Delete products
- Categorize products (Rice, Dal, Cooking Oil)
- Track stock availability

### Stock Management
The system keeps track of product stock levels and helps monitor inventory.

- Category
- Price
- Unit
- Stock availability

## Use Case

This system is designed for **small grocery retail shops** to simplify their daily billing operations and maintain digital sales records.

It helps store owners:
- Generate bills faster
- Monitor sales performance
- Track inventory
- Analyze sales trends
